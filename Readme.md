# Rinoa Analysis Service

## Runing 

### First create a virtual env

```bash
python3 -m venv venv
```


### Second Activate this env

```bash
source venv/bin/activate
```

### Third install dependencies

```bash
pip install -r requirements.txt
```


### Formating, Linting, Type-Checking all in one

```bash
make check-all
```
